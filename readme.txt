Google Plus Badge Widget plugin allows you quickly and easily  to add your Google Plus badge widget in your blog WordPress site.

== Description ==

Google Plus badge widget that is available for free to use. This badge widget provides significant enhancements over the traditional badge widget.

Google Plus Badge Widget plugin allows you to quickly and easily add the Google Plus badge widget to your website. Easy configuration using your Google Plus profile  url, it is adjust style according to your site style.

Google Plus Badge Widget customize settings following features:

* Hide/show google plus cover photo and tagline.
* Set widget color schemes.
* Set widget layout style.
* Set your widget width.
* Check your widget preview on submit data.

Plugin by <a href="http://weblizar.com">Weblizar</a>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `google-plus-badge-widget` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Google Plus Badge Box widget from the widgets window in WordPress' admin panel.
4. Configure the available options with your page type, URL, desired style and size.
